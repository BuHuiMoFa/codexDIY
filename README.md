# TOKENICODE

TOKENICODE 是一款面向中文用户的 AI 桌面工作台。  
它把命令行能力、聊天协作、文件预览、审阅面板、工作区管理和自定义界面整合到一个本地应用里，让用户不用频繁切换终端、编辑器和文件管理器，也能完成日常 AI 协作工作。

## 应用简介

TOKENICODE 适合用来做这些事情：

- 让 AI 帮你处理表格、文档、图片素材和项目文件
- 在同一个窗口里查看处理过程、消息记录和结果文件
- 直接预览生成出来的文件，并快速打开文件位置
- 管理多个工作区、多个会话和不同任务上下文
- 使用自定义 API 供应商，而不是被固定绑定到单一模型服务

它是一个偏“生产力工作台”的桌面软件，不只是简单聊天工具。

## 核心特点

- 多 API 供应商配置
  支持自定义服务商地址、模型和密钥，不强制绑定单一供应商。

- 内置 CLI 运行能力
  应用可直接驱动本地任务流程，减少手动打开命令行的操作成本。

- 多工作区管理
  支持维护多个工作目录，也支持把文件夹拖入应用作为工作区。

- 聊天与任务协作一体化
  可以在聊天界面里发起任务、查看处理中状态、跟踪结果，并继续追加修改要求。

- 文件预览能力
  支持在应用内查看生成文件，并提供“预览”“打开位置”等快捷操作。

- 审阅与变更查看
  支持查看任务过程中的修改内容、差异信息和结果变化，方便核对输出是否符合预期。

- 自定义界面外观
  支持设置背景图片、头像、界面透明度和背景增强模式，兼顾个性化和可读性。

## 适用人群

- 经常用 AI 处理文件和资料的办公用户
- 需要批量生成或修改表格、文档内容的运营人员
- 想把 AI 工具做成桌面软件长期使用的个人用户
- 希望统一管理工作区、会话记录和生成文件的团队成员

## 典型使用场景

- 商品图册整理
  让 AI 按命名规则生成商品表格、分类表、补充说明，并直接预览结果文件。

- 文件批量处理
  在一个工作区中持续处理 Word、Excel、PDF、图片或压缩包相关任务。

- 内容修订与复查
  查看 AI 做了哪些修改，确认新增、删除或替换内容是否正确。

- 项目资料协作
  用多个工作区分别管理不同项目，避免不同任务内容混在一起。

## 使用方式

1. 打开应用后先选择或拖入一个工作区。
2. 在聊天输入框中直接描述你的任务需求。
3. 观察处理过程、思考状态、文件输出和审阅面板。
4. 对结果不满意时，可以继续补充要求、重新编辑或继续迭代。
5. 生成文件后，可在界面中直接预览或打开文件所在位置。

## 设计目标

TOKENICODE 想解决的不是“能不能和 AI 对话”，而是“能不能把 AI 真正用进每天的文件工作流里”。  
因此它更强调：

- 本地桌面体验
- 中文场景适配
- 文件处理闭环
- 可视化任务过程
- 更低的使用门槛

## 当前项目说明

本项目基于桌面端架构持续迭代，当前重点放在这些方向：
